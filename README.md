# ALAB 308.5.1: Creating Reusable Functions

🌐 Live Demo: 🔗 https://tahmina-official.github.io/Creating-Reusable-Functions/

## Overview
This project demonstrates core JavaScript concepts including:

- Functions
- Array methods (`reduce`, `filter`, `map`, `sort`)
- Recursion
- Object manipulation

## Parts Completed

### Part 1
Basic functions:
- Sum array
- Average array
- Find longest string
- Filter strings by length
- Print numbers recursively

### Part 2
Array method practice:
- Sort objects by age
- Filter ages greater than 50
- Rename `occupation` to `job`
- Increment ages
- Calculate total and average age

### Part 3
Object manipulation:
- Increment age directly
- Create copy and increment age
- Add `updated_at`
- Handle object reference vs copy

## Key Concepts
- `reduce()` for combining values
- Recursion without loops
- Spread operator (`...`)
- Object references
- Defensive programming
